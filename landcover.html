Kenya Land Cover Classification

Spatial analysis of land cover distribution using satellite imagery

Project Overview

This project analyzes land cover distribution across Kenya using remote sensing classification techniques.

Methodology

Satellite imagery was processed and classified to identify vegetation, water bodies, and built-up areas.

Tools Used

ArcGIS Pro and remote sensing spatial analysis.
