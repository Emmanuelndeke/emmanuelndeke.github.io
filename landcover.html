Kenya Land Cover Classification

Spatial analysis of land cover distribution using satellite imagery

Project Overview

This project maps land cover distribution across Kenya using remote sensing classification techniques to understand environmental patterns.

Methodology

Satellite imagery was processed and classified to identify major land cover types including vegetation, water bodies, and built-up areas.

Tools Used

ArcGIS Pro, remote sensing classification, and spatial analysis tools.

Key Insights

Vegetation cover dominates western Kenya while arid land dominates northern regions.
